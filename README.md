* Selection Sort
* Bubble Sort
* Merge Sort

## Main Libraries Used :
* Swing Worker
* Swing Components
* Graphics 2D

## Skills Learned :
* Multi-Threading using Swing Worker
* Customising Swing Components
* Graphics 2D
* Apply Sorting Algorithms
